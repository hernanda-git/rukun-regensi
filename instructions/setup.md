# Project Setup

1. Duplicate `.env.example` into `.env` and fill in your Supabase project URL + anonymous key:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
2. Run `npm install` to pull Next.js, Tailwind, NextUI, and Supabase JS dependencies.
3. Launch the app with `npm run dev` and visit `http://localhost:3000`.

This project is intentionally mobile-first, so test it on narrow viewports or devices.
